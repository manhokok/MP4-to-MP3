### Sử Dụng

Tool convert `MP4` to `MP3`
Tạo tệp `mp3` ở thư mục chính sau đó chạy:
```console
sh mp3
```

Tool Quét `MP3`
```console
sh scanmp3
```

Tool Quét `MP4`
```console
sh scanmp4
```
